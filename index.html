c语言输出菱形图案

Share

简介

用“*”输出菱形

思路

 1.把菱形分为上下两部分（上4行下3行）
 2. i 控制循环次数， j 控制空格数， k 控制“*”的个数
 3. 第一行3个空格，第二行2个空格...  （n=4，j=n-1)
    第一行1个“*” ，第二行3个“*”...（k=2*i-1)
   下3行： 
 4. 空格数 1→2→3 
    i--（3→2→1）  4-3，4-2，4-1 为空格数
5.  “*”  2×3-1，2×2-1,2×1-1

代码

#include <stdio.h>

#include <stdlib.h>

int main()
{
int i,j,k,n=4;
for(i=1;i<=n;i++)
{
for(j=n-1;j>=i;j–)
{
printf(“ “);
}
for(k=1;k<=2i-1;k++)
{
printf(““);
}
printf(“\n”);
}

for(i=n-1;i>=1;i–)
{
for( j=n-i;j>0;j– )
{
printf(“ “);
}
for( j=0;j<2i-1;j++ )
{
printf(““);
}
printf(“\n”);
}
return 0;
}

正三角形

Share

简介

用“*”输出三角形

思路

1.每一行的空格数以及“”的个数
2.第一行三个空格 第二行递减 （j=3,j–）
3.第一行一个“”第二行3个“” （k<=i2-1）

代码

#include <stdio.h>
int main() {
int i,j,k;
for(i=1;i<5;i++)
{
    for(j=3;j>=i;j--)
    {
        printf(" ");

    }
    for(k=1;k<=i*2-1;k++)
    {
        printf("*");
    }

printf("\n");
}
return 0;

}

计算1-2+3-4+5-6+…-100的结果

思路

1.算出1-100奇数之和
2.算出1-100偶数之和
3.奇数之和减去偶数之和

代码

#include <stdlib.h>

int main()
{
    int a=0,b=0,sum;
    int count = 0;
    int c=0;
for(int i=1;i<=100;i++)
    {
        if(i%2==0)
        {

        a=a+i;
        }
        else
        {

            b=b+i;
        }
    }
    sum=b-a;
    printf("a=%d\n",a);
