c语言输出菱形图案

Share

简介

用“*”输出菱形

思路

 1.把菱形分为上下两部分（上4行下3行）
 2. i 控制循环次数， j 控制空格数， k 控制“*”的个数
 3. 第一行3个空格，第二行2个空格...  （n=4，j=n-1)
    第一行1个“*” ，第二行3个“*”...（k=2*i-1)
   下3行： 
 4. 空格数 1→2→3 
    i--（3→2→1）  4-3，4-2，4-1 为空格数
5.  “*”  2×3-1，2×2-1,2×1-1

代码

#include <stdio.h>

#include <stdlib.h>

int main()
{
int i,j,k,n=4;
for(i=1;i<=n;i++)
{
for(j=n-1;j>=i;j–)
{
printf(“ “);
}
for(k=1;k<=2i-1;k++)
{
printf(““);
}
printf(“\n”);
}

for(i=n-1;i>=1;i–)
{
for( j=n-i;j>0;j– )
{
printf(“ “);
}
for( j=0;j<2i-1;j++ )
{
printf(““);
}
printf(“\n”);
}
return 0;
}
