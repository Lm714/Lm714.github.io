正三角形

Share

简介

用“*”输出三角形

思路

每一行的空格数以及“*”的个数

代码

#include <stdio.h>
int main() {
int i,j,k;
for(i=1;i<5;i++)
{
    for(j=3;j>=i;j--)
    {
        printf(" ");

    }
    for(k=2;k<=i*2;k++)
    {
        printf("*");
    }

printf("\n");
}
return 0;

}
