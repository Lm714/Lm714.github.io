简介

计算1-2+3-4+5-6+…-100的结果

思路

1.算出1-100奇数之和
2.算出1-100偶数之和
3.奇数之和减去偶数之和

代码

#include <stdio.h>
#include <stdlib.h>

int main()
{
    int a=0,b=0,sum;
    int count = 0;
    int c=0;
for(int i=1;i<=100;i++)
    {
        if(i%2==0)
        {

        a=a+i;
        }
        else
        {

            b=b+i;
        }
    }
    sum=b-a;
    printf("a=%d\n",a);
    printf("b=%d\n",b);
    printf("sum=%d\n",sum);
    return 0;
}
